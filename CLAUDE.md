# PA Koovis — Repository Context

## What This Repo Is

PA Koovis (Personal Assistant by Koovis AI) is the intelligent layer above all Koovis AI systems. This repo will hold the codebase for the PA — intent routing, context engine, integrations, voice pipeline, and messaging interfaces.

## Current State
